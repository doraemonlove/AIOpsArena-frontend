```mermaid
flowchart TD
    start([函数调用])
    getSE[获取startTime和endTime]
    firstInCheck{第一次进入}
    return([return true])
    isNumberCheck{startTime和endTime是否是数字}
    changeToNumber[转换为数字]
    notNumber([重定向到to.name并将
    startTime赋值为this.__startTime.value
    endTime赋值为this.__endTime.value
    ])
    stCheck{startTime是否大于此时或者小于等于0}
    etCheck{endTime是否大于此时或者小于等于0}
    rangeCheck{时间区间是否大于允许的最大时间区间或者小于0}
    stChange([重定向到to.name并将startTime赋值为前一分钟])
    etChange([重定向到to.name并将endTime赋值为现在])
    stetChange([重定向到to.name并将startTime赋值为前一分钟，endTime赋值为现在])
    yes[将startTime赋值给this.__queryStartTime.value,将endTime赋值给this.__queryEndTime.value]

    start --> getSE
    getSE --> firstInCheck
    firstInCheck -- 是 --> return
    firstInCheck -- 否 --> isNumberCheck
    isNumberCheck -- 是 --> changeToNumber
    isNumberCheck -- 否 --> notNumber
    changeToNumber --> stCheck
    stCheck -- 是 --> stChange
    stCheck -- 否 --> etCheck
    etCheck -- 是 --> etChange
    etCheck -- 否 --> rangeCheck
    rangeCheck -- 是 --> stetChange
    rangeCheck -- 否 --> yes
    yes --> return
```
